MobileGameWebAPI[ASP.NETwebAPI] - пример серверной части мобильной игры. Используется ASP.net и PostgreSQL
RTS_game_server[.NET Core] - ранняя версия серверного кода стратегии Mars2000, написанная на .net core (хостился на linux)
SpaceShooter[Unity3d] - тестовое задание, игра космический шутер, написанная на Unity
simplepinball[Unity3d] - тестовое задание, игра пинбол для андройда, 
	она же в топе магазина амазон по запросу pinball(называется simple pinball full): https://www.amazon.com/s/field-keywords=pinball
	она же в google play: https://play.google.com/store/apps/details?id=com.vsibhekzn.simplepinball.full&hl=ru
bin - скомпилированные игры
----------------------------
В папке Examples:
mmoAPI.txt – кусок серверной части mmo игры для gamesparks.com. Язык JavaScript
MMOLocation.ts – кусок серверной части, скрипт для для сессии реального времени в mmo игре на gamesparks.com. Язык TypeScript (все в одном файле - это ограничение gamesparks.com)
RTSFogOfWar.shader - шейдер тумана войны для игр на Unity3d.
WebApi.ts,ObjectModel.ts - пример TypeScript на веб сайте
